# smdr
Station Message Detail Recording ( SMDR ) for PBX NEC SV8300 
<br />
-    **smdr_interact.py** - console daemon. Help to trace missing configuration.
-    **smdr_service.py**  - Windows service.
<br />
    Required modules: 
<br />
        mysql_connector >= 8.0.16
<br />
        protobuf >= 3.8.0
<br />
        pywin32 >= 2.2.4
<br />
        setuptools >= 41.0.1
<br />
        six >= 1.12.0
